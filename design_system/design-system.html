Card Structure
.card .stretch .stretch-full
Cards use .stretch and .stretch-full to fill their column height. Card sections: .card-header, .card-body, .card-footer. Action area: .card-header-action with .card-header-btn.
Motion & Interaction
Sidebar Transitions

The sidebar supports collapsed/expanded states via #menu-mini-button / #menu-expend-button. Toggling adds/removes a class on the body, and the sidebar transitions width with CSS.

Dropdown Animation

Dropdowns use Bootstrap's built-in transitions. The theme applies smooth opacity/transform transitions on .dropdown-menu.

Dark/Light Theme Toggle

Use the / toggle in the header to switch themes. All colors, backgrounds and borders transition smoothly between modes.
